# Ассистент Лекций MVP

Мобильное приложение для записи лекций с автоматической транскрибацией, генерацией конспектов и умным чатом по содержимому.

## Возможности

- **Запись аудио** — записывайте лекции прямо на телефон
- **Распознавание речи (ASR)** — транскрибация с помощью faster-whisper
- **Мультиязычность** — поддержка русского, казахского и английского
- **Структурированные конспекты** — автоматическая генерация заметок с темами, определениями и фактами
- **RAG-чат** — задавайте вопросы и получайте ответы только на основе лекции с таймкодами


## Быстрый старт

### Требования

- Python 3.11+
- Flutter 3.x
- Gemini API ключ (бесплатный) или OpenAI API ключ

### 1. Запуск бэкенда

```bash
cd backend

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или: venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env и добавьте GEMINI_API_KEY

# Запуск сервера
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Запуск мобильного приложения

```bash
cd frontend

# Установка зависимостей
flutter pub get

# Запуск на устройстве/эмуляторе
flutter run
```

### 3. Проверка

- API документация: http://localhost:8000/docs
- Health check: http://localhost:8000/health

## Технологии

**Бэкенд:**
- FastAPI — веб-фреймворк
- faster-whisper — распознавание речи
- sentence-transformers — эмбеддинги для RAG
- ChromaDB — векторное хранилище
- Google Gemini / OpenAI — генерация текста (легко переключается)

**Фронтенд:**
- Flutter — мобильный фреймворк
- record — запись аудио
- dio — HTTP клиент

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/lectures/upload` | Загрузка аудио |
| GET | `/api/lectures` | Список лекций |
| GET | `/api/lectures/{id}` | Детали лекции |
| GET | `/api/lectures/{id}/transcript` | Транскрипт |
| GET | `/api/lectures/{id}/summary` | Конспект |
| POST | `/api/lectures/{id}/chat` | RAG-чат |
